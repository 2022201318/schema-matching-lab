3、将表级别的json投入lightrag，结合上一步输出的数据域的信息，依次输出每个数据域内可能又匹配关系的表。
输出的内容都是正确的，是不是完整的还需要看最后的结果
4、将列级别的json投入lightrag，结合上一步输出的可能匹配表的信息，依次输出每个匹配表内可能又匹配关系的列。
粗看查全率应该是很高的
